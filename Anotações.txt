		#Comandos estudados durante a aula sobre o Git/Github:

Git clone:	Clona um repositório (faz uma cópia idêntica da versão mais recente 
de um projeto em um repositório e a salva em seu computador)


Cd:		Exibir o "diretório de trabalho atual"


Git status:		Obter o status completo do repositório (nos dá todas as 
informações necessárias sobre a branch atual)


Dir:		Listar o conteúdo do diretório de trabalho atual.


Git add .:	Adiciona todas alterações no diretório ativo à área de atuação 
(inclui as alterações de um ou vários arquivos em nosso próximo commit).


Git commit:	 Salva suas alterações no espaço de trabalho local (ao usar -m, podemos
deixar um comentário) Ex.: git commit -m "mensagem do commit"


Git push <repositório-remoto> <nome-da-branch>:		Faz o upload dos seus commits 
no repositório remoto.


Git config --global user.email/Git config --global user.name:	Permite ver e atribuir 
variáveis de configuração que controlam todos os aspectos de como o Git aparece e opera.
